### 1. float

: 요소를 좌, 우 방향으로 띄움

- block 요소를 인라인 방식으로 띄우기 위해 사용

```html
<style>
  .box {
    width: 150px;
    height: 100px;
    background-color: tomato;
    color: white;
    font-size: 30px;
    margin: 10px;
    float: left;
  }

  .clearfix::after {
    content: "";
    clear: both;
    display: block;
  }

  .box1 {
    width: 200px;
    height: 200px;
    background-color: violet;
  }
</style>

<body>
  <div class="clearfix">
    <div class="box">1</div>
    <div class="box">2</div>
    <div class="box">3</div>
  </div>
  <div class="box1">4</div>
</body>
```

```
💡 inline-block vs float
1) inline-block 
    - 하단 공백 제거 
      - vertical-align or font-size:0 (글자가 존재한다면 이건 못씀)
    - 태그 다음에 있는 엔터 때문에 요소 사이에 공백 생김 
      - font-size:0 or 태그 사이에 엔터 없애기
    - text-align 사용 가능

2) float
    - 모든 자식들이 float 된 상태이면 부모는 자식요소인 줄 모름
      - overflow:hidden
      - clearfix 클래스 추가 후 작업
      - inline-block 정렬 개념과 다르게 오른쪽 배치 가능
      - 요소들을 왼쪽, 오른쪽으로 나눠서 배치 가능
```

```css
/* 인라인 블럭 */
.grid {
  width: 500px;
  margin: 0 auto;
  background: white;
  /* overflow: hidden; */
  font-size: 0;
  text-align: right;
}
.grid-item {
  width: 100px;
  height: 100px;
  display: inline-block;
  vertical-align: top;
}

/* float */
.grid-item {
  width: 100px;
  height: 100px;
  float: right;
}
.clearfix::after {
  content: "";
  display: block;
  clear: both;
}

<div class="grid clearfix">
  <div class="grid-item a"></div>
  <div class="grid-item b"></div>
  <div class="grid-item c"></div>
</div>
```

### 2. 자식 요소 float 사용

- 자식 요소 float 사용 시 부모는 자식으로 인식 x
    - 해결 방법

          1) overflow:hidden → float 이후의 다른 요소들이 제자리를 찾지 못함

          2) clear: both → float 해제

```html
<style>
  body {
    background-color: #ddd;
  }
  .box {
    background-color: white;
    overflow: hidden;
		/* height 지정 */
  }
  .left {
    float: left;
  }
  .right {
    float: right;
  }
  .clearfix {
    clear: both;
  }
</style>

<body>
  <div class="box">
    <img src="../img/left-arrow.png" alt="" class="left" />
    <img src="../img/right-arrow.png" alt="" class="right" />
    <div class="clearfix">Awesome</div>
  </div>
</body>
```

### 5. List float

```html
<style>
  * {
    margin: 0;
    padding: 0;
  }
  .portfolio {
    width: 960px;
    margin: 100px auto;
    background-color: black;
  }
  .portfolio li {
    list-style-type: none;
    float: left;
    width: 225px;
    height: 225px;
    background-color: rgb(52, 119, 124);
    margin-right: 20px;
  }
  .portfolio li:last-child {
    margin-right: 0;
  }
  .clearfix::after {
    content: "";
    clear: both;
    display: block;
  }
</style>

<body>
  <ul class="portfolio clearfix">
    <li>pic1</li>
    <li>pic2</li>
    <li>pic3</li>
    <li>pic4</li>
  </ul>
</body>
```
